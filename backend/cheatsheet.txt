# MindEase Local Development Cheat Sheet (continued)
# ----------------
# 3️⃣ Start Frontend (React)
# ----------------
# Open a NEW terminal (do not stop backend)
# Navigate to frontend folder
cd ../frontend

# Install dependencies (only first time or if new packages added)
npm install

# Start frontend dev server
npm start

# The frontend should now run at:
# http://localhost:3000
# The backend API is running at:
# http://127.0.0.1:8000

# ----------------
# 4️⃣ Helpful Tips
# ----------------
# - Keep backend running in one terminal, frontend in another.
# - If you update frontend code (e.g., src/App.js), React hot reload should show changes automatically.
# - If you update backend code, uvicorn --reload will automatically restart the server.
# - Common issues:
#   * Port conflicts: Use a different port with --port <number>
#   * Virtual environment not activated: backend commands may fail.
#   * Node/npm errors: Make sure Node.js is installed globally and version >= 18.
# - To stop any server: Press CTRL+C in that terminal.
# - To reactivate virtual environment after closing terminal: repeat Step 1.
# - You can test API endpoints using curl or Postman:
#     curl http://127.0.0.1:8000/your_endpoint
